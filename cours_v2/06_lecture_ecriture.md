# Lecture et écriture

## 1. Lecture

Récupérer une valeur provenant de l'extérieur (clavier).

Instruction : **LIRE**

---

## 2. Écriture

Afficher une valeur à l'écran.

Instructions : **AFFICHER**, **AFFICHERCALCUL**

- **AFFICHER** : affiche un texte ou la valeur d'une variable
- **AFFICHERCALCUL** : affiche le résultat d'un calcul

---

## 3. Exemple

```
FONCTIONS_UTILISEES
VARIABLES
  A EST_DU_TYPE NOMBRE
  B EST_DU_TYPE NOMBRE
DEBUT_ALGORITHME
  LIRE A
  LIRE B
  AFFICHERCALCUL A+B
FIN_ALGORITHME
```
